- ✕ 닫기 → `localStorage`에 dismiss 시각 저장

**시각**
- z-index 50으로 헤더보다 위
- 모바일에서는 한 줄, 데스크톱에서는 부가 설명 ("· 빅데이터 기반...") 표시

## 슬라이드 11 · 히어로 슬라이더 (6슬라이드)

**왜 만들었는가**
- 첫 방문자에게 사이트의 5가지 핵심 도구 + 교재를 한 번에 노출

**6 슬라이드 구성**
1. **독끝 전기기능사 필기** (검정/앰버 그라디언트) — 빅데이터 기반 교재 셀링
2. **CBT 모의고사** (블루) — "합격까지, 한 곳에서"
3. **기출 해설강의** (에메랄드) — 무료 공개 예정
4. **플립 암기카드** (바이올렛/핑크) — 300장 핵심
5. **오디오북** (앰버/오렌지) — 출퇴근길 합격
6. **이론 시뮬레이터** (인디고/퍼플) — 직접 만져보기

**구현**
- 5.5초 자동 회전 (호버 시 일시정지)
- 좌·우 화살표 (sm:이상)

- 진행 바 (애니메이션)
- 큰 이모지 박스 (6초 주기 부유 애니메이션)
- 각 슬라이드: 뱃지 / 제목 / 강조 / 설명 / Primary CTA / Secondary CTA

## 슬라이드 12 · 합격 로드맵 — 3단계 (`<HowItWorks>`)

**왜 만들었는가**
- 학습 사이트의 "어떻게 합격하지?"를 한 번에 보여주는 핵심 narrative
- 교재 → 모의고사 → 부가서비스 흐름을 명시화

**구현**
- **STEP 01 — 기본 학습** (앰버) · 📘 독끝 전기기능사 필기 교재
- **STEP 02 — 실전 콘텐츠** (블루) · 🖥 CBT 모의고사
- **STEP 03 — 부가 서비스** (바이올렛) · ✨ 오디오북·플립카드·시뮬

**시각 특징**
- 거대 ROADMAP 워터마크 텍스트 (배경에 4% 투명도)
- 각 카드 우상단에 거대 단계 번호 (140px)
- 컬러 보더 + 호버 lift + 그림자
- 카드 사이 그라디언트 라인으로 단계 연결
- 헤더 헤드라인: "합격까지, **단 3단계**" (앰버 형광펜 효과)

**기술** · `src/components/HowItWorks.tsx`

**캡처 가이드** — 섹션 풀샷 + 단계 카드 1개 줌인
